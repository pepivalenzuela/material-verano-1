📖 TEORÍA

En este módulo combinamos todos los contenidos anteriores para resolver problemas completos. Los dos nuevos conceptos que introducimos son la mediana de un triángulo (recta que une un vértice con el punto medio del lado opuesto) y el baricentro (punto donde se cortan las tres medianas).

📐 1. Medianas de un triángulo

La mediana desde un vértice es la recta (o segmento) que une ese vértice con el punto medio del lado opuesto. Un triángulo tiene tres medianas.

Mediana desde \\(A\\) → une \\(A\\) con el punto medio de \\(BC\\)
Ejemplo resuelto 1 — Ecuación de una mediana
①Triángulo \\(A=(0,0)\\), \\(B=(6,0)\\), \\(C=(3,6)\\)
②Punto medio de \\(BC\\): \\(M_{BC}=\\left(\\dfrac{6+3}{2},\\dfrac{0+6}{2}\\right)=(4.5,3)\\)
③Mediana desde \\(A=(0,0)\\) a \\(M_{BC}=(4.5,3)\\): pendiente \\(m=\\dfrac{3}{4.5}=\\dfrac{2}{3}\\)
④Ecuación: \\(y=\\dfrac{2}{3}x\\) (pasa por el origen)

📐 2. Baricentro

El baricentro \\(G\\) es el punto de intersección de las tres medianas. Se puede calcular de dos formas:

Fórmula directa: \\(G=\\left(\\dfrac{x_A+x_B+x_C}{3},\\;\\dfrac{y_A+y_B+y_C}{3}\\right)\\)

Intersección: resolver el sistema formado por dos medianas cualesquiera
Ejemplo resuelto 2 — Baricentro por fórmula
①\\(A=(0,0)\\), \\(B=(6,0)\\), \\(C=(3,6)\\)
②\\(G=\\left(\\dfrac{0+6+3}{3},\\dfrac{0+0+6}{3}\\right)=(3,2)\\)
✓Verificación: \\(G=(3,2)\\) en \\(y=\\dfrac{2}{3}x\\): \\(\\dfrac{2}{3}\\cdot3=2\\) ✓
Ejemplo resuelto 3 — Baricentro por intersección de medianas
①Mediana desde \\(A\\): \\(y=\\dfrac{2}{3}x\\)
②Mediana desde \\(B=(6,0)\\) a \\(M_{AC}=(1.5,3)\\): \\(m=\\dfrac{3-0}{1.5-6}=-\\dfrac{2}{3}\\). Ecuación: \\(y=-\\dfrac{2}{3}x+4\\)
③Sistema: \\(\\dfrac{2}{3}x=-\\dfrac{2}{3}x+4\\Rightarrow\\dfrac{4}{3}x=4\\Rightarrow x=3,\\;y=2\\)
④Baricentro: \\(G=(3,2)\\) ✓
💡 Propiedad del baricentro
El baricentro divide cada mediana en razón \\(2:1\\) desde el vértice. Es el "centro de gravedad" del triángulo.

📐 3. Estrategia para problemas completos

La mayoría de los problemas de geometría analítica siguen estos pasos:

  1. Identifica los datos: puntos, rectas, condiciones.
  2. Calcula lo que necesitas: vectores, puntos medios, pendientes.
  3. Escribe las ecuaciones de rectas implicadas.
  4. Resuelve sistemas para encontrar intersecciones.
  5. Verifica sustituyendo los resultados.
